# Commands
Tool for storing commands (or whatever) from command line and querying them by tags.

# Usage
```
Add entry:
python commands.py --add "command" --comment "what the command does" --tags "tag1 tag2 tag3"

Find entries:
python commands.py --find "tag1 tag2"
````
